# Edison Engineering API

Backend API for solar & electrical services website.

## 🚀 Tech Stack
- .NET Core Web API
- SQL Server
- Entity Framework Core
- Clean Architecture

## 📂 Project Structure
- EdisonEngineering.API → Entry point (Controllers)
- EdisonEngineering.Application → Business logic
- EdisonEngineering.Domain → Entities
- EdisonEngineering.Infrastructure → DB & external services

## ⚙️ How to Run
1. Navigate to API project:
   cd EdisonEngineering.API

2. Run the project:
   dotnet run

3. Open Swagger:
   http://localhost:5209/swagger
